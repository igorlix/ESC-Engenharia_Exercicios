Sobre o Sistema
Análise de Sentimento

Sistema web desenvolvido para análise automática de sentimento em textos utilizando técnicas de processamento de linguagem natural (NLP) e inteligência artificial.

Funcionalidades
  • Análise de sentimento em português e inglês
  • Classificação: positivo, negativo ou neutro
  • Pontuação numérica de -1.0 a 1.0
  • Histórico de análises por usuário
  • Interface moderna com tema escuro
Métodos de Análise

O sistema utiliza uma abordagem de três camadas para garantir precisão:

1. Amazon Bedrock (LLM)

Método principal utilizando o modelo Amazon Nova Micro via AWS Bedrock para análise contextual avançada. Proporciona maior precisão ao entender nuances da linguagem.

2. Dicionário Customizado

Fallback com dicionário de 60+ palavras em português e inglês. Analisa presença de palavras positivas e negativas para calcular o sentimento.

3. TextBlob

Biblioteca de NLP utilizada quando não há palavras-chave identificadas no dicionário. Fallback final para garantir que todas as análises sejam processadas.

Tecnologias
